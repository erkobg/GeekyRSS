GeekyRSS.

Android application for following RSS feeds, with possibility to read articles in offline mode

Target API : 8 ( Android 2.2 Froyo)

Activities : 5

Services : 1 (background job to check old articles)

Time used: several hours (1-2) for 7 days, just when I had the chance

Description

When the application starts, the user is able to see a list of all the feeds that he subscribed to along with the number of read and unread news for each feed. The user is able to subscribe to a new feed by providing its URL as well as unsubscribe from existing feeds. In the context menu there is add new feed, and when "Long hold" on the specific Feed - there comes context menu with Sync and Delete option. Syncing of the news is done automatically when the initial screen is loaded if the application finds that there is network (Wi-Fi or Mobile data) connection - you can see progress in the progress bar in the bottom of the Feeds List activity. All the feeds (title, text, URL to the real news, picture and any additional data) are stored in a database for offline reviewing.

When the user clicks on a feed from the initial screen, all the news items that it contains are presented on a new screen in a way that the new (unread) ones can be distinguished from the old - readed ones are Strikethrough styled. A news item is considered old if it is opened. The user is also able to look at a list of featured news from all the feeds that they have subscribed to. Featured news is one that contains a key word or a sequence of words in its title or summarized text. The user is able to manipulate, i.e. add or delete, the list of key words. This is done from a dedicated screen.

There is a background job that regularly checks if the database contains too many entries and that cleans up the oldest ones (in our case - the older than 10 days).

Permissions: android.permission.WRITE_EXTERNAL_STORAGE - used for SQLite database creation

android.permission.ACCESS_NETWORK_STATE - used for checking whether there is internet connection in order to sync

android.permission.INTERNET - used to sync Feeds and Articles
